# Role: 资深前端架构师 (Vue.js 专家)

## Task: 
请基于以下功能需求，使用 **Vue 3 (Composition API) + Vite + Element Plus + Tailwind CSS** 构建一个“应急物资调度管理系统”的前端核心架构代码。

## 核心技术要求：
1. **布局方案**：必须使用 **CSS Grid** 构建大框架（Layout），确保页面元素在不同屏幕尺寸下具备良好的**响应式**表现（适配移动端、平板、PC）。
2. **状态管理**：使用 **Pinia** 管理用户信息、JWT Token 及全局库存预警状态。
3. **网络层**：封装 Axios 拦截器，处理请求头的 JWT 注入、响应拦截以及身份过期（401）的跳转逻辑。
4. **组件化标准**：代码需高度组件化，采用 TypeScript 进行类型定义。

## 模块细化需求：
1. **模块 1 (Auth)**：实现登录页及 RBAC（基于角色的访问控制）侧边栏导航。根据角色（救援人员、仓库管理员、系统管理员）动态过滤路由和菜单。
2. **模块 2 (Inventory)**：物资列表页。包含多条件搜索过滤、出入库操作弹窗、以及使用 Element Plus Tag 展示动态库存状态（如：充足、待补货、严重短缺）。
3. **模块 3 (Dispatch)**：需求审核工作流界面。展示申请列表，点击审核后展示分配建议逻辑，并模拟生成配送任务的反馈。
4. **模块 4 (Logistics)**：物流追踪组件。需预留高德地图 (AMap) 接口位置，实现一个展示车辆实时位置与运输轨迹的可视化容器。
5. **模块 5 (Alert)**：全局通知系统。当库存低于 10% 或物资即将过期时，在顶部导航栏实现一个红色预警气泡通知组件。
6. **模块 6 (Dashboard)**：使用 **ECharts** 生成响应式看板，包含物资消耗趋势折线图和库存占比饼图。

## 输出要求：
- 请先给出建议的项目目录结构。
- 重点编写核心布局文件 `MainLayout.vue`（体现 CSS Grid 响应式逻辑）。
- 编写响应式 `Dashboard.vue` 页面原型。
- 提供权限路由拦截（Router Guard）的核心逻辑代码。
