# Emergency Material Dispatch System - Frontend

## Setup & Run

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Features

- **Responsive Layout**: CSS Grid based layout with a collapsible sidebar for mobile.
- **Role-Based Access Control (RBAC)**: Dynamic menu generation based on user roles (Admin, Warehouse, Rescue).
- **Interactive Dashboard**: Real-time data visualization using ECharts.
- **Inventory Management**: Track material status with visual indicators.
- **Dispatch Workflow**: Manage and authorize dispatch requests.
- **Global Alerts**: Real-time stock warnings in the navigation bar.

## Tech Stack

- Vue 3 + TypeScript
- Vite
- Pinia (State Management)
- Element Plus (UI Components)
- Tailwind CSS (Utility Styling)
- ECharts (Data Visualization)
